# 사부작 앱 프로젝트

## 앱 개요
어린이(만 5~9세) 루틴 습관 형성 + 가상 용돈 리워드 웹앱

## 기술 스택
- Frontend: React + TypeScript + Tailwind CSS
- Backend: Supabase (DB + Auth)
- 배포: Vercel

## 화면 구조
1. 로그인 화면 (부모 계정)
2. 아이 홈 화면 — 아침/저녁 루틴 카드 선택
3. 루틴 상세 화면 — 필수/엑스트라 3열 아이콘 그리드, 스크롤 가능
4. 부모 설정 화면 — 자녀 프로필, 루틴 편집, 포인트 설정, 알림, 계정
5. 통계 화면 — 주간/월간/연간 그래프

## 핵심 규칙
- 아이 화면: 아이콘 + 텍스트 병행, 큰 터치 영역
- 부모 승인 구조: 아이 체크 → 부모 승인 버튼 → 포인트 적립
- 가상 포인트만 사용 (실제 금융 연동 없음)
- 색상: 파스텔 계열 (민트, 노랑, 코랄, 스카이블루)
- 캐릭터: 카테고리별 동물 (🐻 아침, 🐰 위생, 🦊 학습, 🐥 집안일, 🐱 저녁)

## 코딩 스타일
- TypeScript 사용
- 컴포넌트는 src/components 폴더에 분리
- 페이지는 src/pages 폴더에 분리
